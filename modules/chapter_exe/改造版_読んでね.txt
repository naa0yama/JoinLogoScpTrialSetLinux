// chapter_exe の改造版 by Yobi
//

無音＆シーンチェンジ検索chapter_exe（作者のruさんに感謝）を
自動処理で精度を上げるため改造したバージョンです。

■オリジナルからの変更内容

・シーンチェンジ位置精度向上のため動き検出アルゴリズム改修
・１つの無音期間にシーンチェンジが複数あれば２つまで出力
・シーンチェンジ直後に加え、直前のフレーム番号を出力
  「SCPos:直後 直前」の形でフレーム番号を出力します。
  （インターレース片側変化のフレームを自動処理で避けるため追加）
・最終フレーム番号も出力
  （最終フレームをシーンチェンジ扱いするため追加）


これ以外はオリジナルのchapter_exeを参照してください。
改造追加部分についても改変時はソース公開お願いします。

無音＆シーンチェンジ検索chapter_exe（作者のruさんと上記のYobiさんに感謝）を
無音エンドカード終了地点検出用に改悪したバージョンです。

■改造版からの変更内容

・指定フレーム以上無音期間があれば無音中でもシーンチェンジを検出
オプション追加　-b 無音検索数
-b 省略時は60



//
// chapter_exe の改造版mod3 by Yobi
//

無音＆シーンチェンジ検索chapter_exe（作者のruさん、上記のsysuzuさんに感謝）を
シーンチェンジ検索フレーム位置調整したバージョンです。
本来のシーンチェンジ位置が検索対象外フレームとなる場合があり修正。

■mod3の変更内容
・シーンチェンジ検索を行う無音フレーム前後の拡張
  オプション追加　-e 拡張フレーム数
  （-e 省略時は1で前後1フレームずつ拡張。0の時は拡張しない）
・無音シーンチェンジ検索フレーム位置関連の修正
・長時間無音は連続的に無音シーンチェンジ検出した上で-bオプション適用のため
  出力結果が少し変わります（境界つなぎを気にしないで済むように変更）



//
// chapter_exe の改造版mod4 by Yobi
//
不要な誤検出を減らして必要な検出を増やすなど、シーンチェンジ位置検出ミスを
減らすための改修をしたバージョンです。

■mod4の変更内容
・シーンチェンジ判定アルゴリズムを改修
  シーンチェンジ判定に従来より細かい制御を追加。
・長時間無音で画像変化のない位置を出力をしない修正
・音声入力に"lwinput.aui"を使用かつFAWを使わない時は、無音確認に間引きを行わず
  常時連続フレーム読み実行（連続フレーム読み出しが早く安定しているため）
  オプション"--thin"指定時は強制間引き、"--serial"指定時は強制連続にする
・画像変化なしと判定した無音シーンチェンジ位置出力にはマーク"＿"を付加

[現在のマーク一覧]
 ★：間隔15秒  ★★：間隔30秒  ★★★：間隔45秒  ★★★★：間隔60秒
 ＠：第2候補   ○：長時間無音による候補  ＿：画像変化なしの候補
